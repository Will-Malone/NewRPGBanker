Assigning a variable to `this` instead of properly using arrow lambdas may be a symptom of pre-ES6 practices
or not managing scope well.

## Examples

<!--tabs-->

### ‚ùå Incorrect

```js
const self = this;

setTimeout(function () {
  self.doWork();
});
```

### ‚úÖ Correct

```js
setTimeout(() => {
  this.doWork();
});
```
